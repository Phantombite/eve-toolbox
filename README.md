# EVE Toolbox

EVE Online Companion Tool für Multiboxer und Multi-Account-Nutzer.

## Windows — Direkt starten (kein Install nötig)
Doppelklick auf `start_windows.exe`

## Windows — Fallback (Python erforderlich)
Doppelklick auf `start_windows_fallback.bat`

## Linux
```bash
bash start_linux.sh
```

## Entwickler
Build-Tools liegen im `build/` Ordner.
```bash
pip install -r eve_toolbox/requirements.txt
python eve_toolbox/main.py
```
